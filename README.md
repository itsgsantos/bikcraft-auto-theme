**BC AutoTheme** is a modern landing page project focused on advanced CSS techniques and seamless user experience. The core feature is its ability to automatically synchronize with the user's operating system theme (Dark or Light) without a single line of JavaScript. 

Inspired by professional boutique automotive branding, it serves as a showcase of clean code, responsive design, and modern web standards.

## 🚀 Technical Features

* **Native Auto-Theming:** Advanced implementation of the `prefers-color-scheme` media query for effortless theme switching.
* **Modern CSS Variable Architecture:** Use of CSS Custom Properties to manage color palettes, ensuring a maintainable and scalable codebase.
* **Pixel-Perfect Responsiveness:** A fluid layout built with **CSS Grid** and **Flexbox** that adapts gracefully across all breakpoints.
* **Performance Focused:** Zero external libraries or heavy assets, resulting in lightning-fast load times and high Core Web Vitals scores.
* **Semantic & Accessible:** Built with HTML5 best practices to provide a consistent experience for all users and assistive technologies.

## 🛠️ Tech Stack

- **Markup:** HTML5 (Semantic & Structured)
- **Styling:** CSS3 (Variables, Grid, Flexbox, Animations)
- **Methods:** Mobile-first workflow & `prefers-color-scheme` logic
- **Deployment:** Vercel

## 🧠 Key Learnings

Developing this project was a deep dive into **native CSS power** and **system-level integration**.

**Main Takeaways:**
- Mastering **CSS Custom Properties** to switch entire color schemes by changing a few variables.
- Implementing **smooth transitions** that respect user's system preferences.
- Understanding the synergy between **CSS Grid and Flexbox** for complex responsive modules.
- Perfecting visual hierarchy using only CSS-based spacing and typography rules.

## 💻 Getting Started

### Prerequisites
A modern web browser (supporting CSS Variables and `prefers-color-scheme`).

### Local Execution
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/itsgsantos/bc-autotheme.git](https://github.com/itsgsantos/bc-autotheme.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd bc-autotheme
    ```
3.  **Run the project:**
    Simply open `index.html` in your browser.
